# Security Policy

This project is under active development. The latest version is supported.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

- Do NOT create a public issue
- Email: rajeev rajeevsharmamachphy@gmail.com


We will respond within 48 hours.

- Dashboard

- Configuration files
